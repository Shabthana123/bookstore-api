# Bookstore API

A RESTful API built with **Spring Boot** for managing books in a bookstore. This project demonstrates CRUD operations, custom queries using **Spring Data JPA**, and integration with **MySQL**.

---

## Features

- **CRUD Operations**: Create, Read, Update, Delete books.
- **Custom Queries**: Filter books by year of publication and type.
- **Spring Data JPA**: Simplified database interactions.
- **MySQL Integration**: Persistent storage for books.
- **REST Endpoints**: Easy integration with frontend applications.

---

## Technologies Used

- Java 17
- Spring Boot 3
- Spring Data JPA
- MySQL
- Maven
- Tomcat Embedded Server
